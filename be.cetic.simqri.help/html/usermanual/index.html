Welcome !

Welcome to the Help Section of the SimQRi Eclipse Desktop Application. This local website will help you to find and to learn all you need to know about the usability of this application. In summary :

  • Information about graphical elements and their properties
  • How to initialize a new SimQRi Project
  • Functionalities of the graphical modeling tools
  • Management of the queries edition table (UPDATED)
  • How to write a correct query (NEW)
  • How to set and execute your simulations (UPDATED)
  • How to check and analyse simulations results (UPDATED)

SimQRi is a CORNET project in partnership with the Fraunhofer Institute, which aims at quantifying the risks induced by procurement of supplies for complex factory chains and manage their impact in manufacturing processes.
