One Shot Simulation Results

Once the "One Shot" simulation is over, results are displayed in a new window. This window is structured in several tabs :

  • Trace (Events) : Details each action of the simulation at a defined time unit
  • Elements : shows the final state of each property for every elements
  • Queries : shows queries results

Save as (One Shot Simulation)

You can also save your results in a text file with a click on the "Save as" button. The file extension is already set by the system. All you have to do is to specify the name and the location of the file.

Monte-Carlo Simulation Results

Once the "Monte-Carlo" simulation is over, results are processed through a report template and then generated into files whose the extension was selected by the user before the beginning of the simulation. These files are situated into the "Generated Reports" folder. The name of subfolders are composed by the date and time of its reports generation.
Warning : you may have to do a refresh of the SimQRi project to display the generated report ! (right-click on the name of the project in the workspace and select "refresh").

At the end of the simulation, queries results are also dynamically encoded in the Queries Table :

Monte-Carlo Default Report Templates

The SimQRi Eclipse desktop client provides two default report templates

  • montecarlo-tables report template :

    This kind of report contains 3 sections :
    • Runtime
    • Elements : shows the final state of each property for every elements (max, min, mean and variance)
    • Queries : shows queries results (max, min, mean and variance)
    The following example has been generated using Microsoft Excel format :



  • montecarlo-charts-lite report template :

    This kind of report contains 2 sections :
    • Risk Distribution : a pie chart that shows the importance of each type of risk as a result of the simulation
    • System Risks Histograms : 3 Bar charts that display the mean according to the frequency of each system risk. These values are calculated using system queries during the simulation
    N.B. A more complete graphical report template will be available with the next SimQRi version and will replace this "lite" template...
