One Shot Simulation Results

Once the "One Shot" simulation is over, results are displayed in a new window. This window is structured in several tabs :

  • Trace (Events) : Details each action of the simulation at a defined time unit
  • Elements : shows the final state of each property for every elements
  • Queries : shows queries results


Save as (One Shot Simulation)

You can also save your results in a text file with a click on the "Save as" button. The file extension is already set by the system. All you have to do is to specify the name and the location of the file.

Monte-Carlo Simulation Results

Once the "Monte-Carlo" simulation is over, results are processed through a report template and then generated into files whose the extension was selected by the user before the beginning of the simulation. These files are situated into the "Generated Reports" folder. The name of subfolders are composed by the date and time of its reports generation.
Warning : you may have to do a refresh of the SimQRi project to display the generated report ! (right-click on the name of the project in the workspace and select "refresh").

Monte-Carlo - Overall charts report template

This report template consists in a global view of the risks impact on the elements of the model. It contains several global charts without any detailed results.

This kind of report contains 5 sections :

  • Risk Distribution: a pie chart that represents the share of each kind of risk (quantity, quality, delay) calculated through the queries written by the user.

  • System Risks Histograms: histograms that represent the frequency of each system risk (determined by its corresponding system query) depending on their calculated mean value.

  • Suppliers/Processes/Storages Overall Risks Histograms: histograms that represent an overall of the frequency of each quantity/quality/delay risks that affect suppliers/processes/storages of the model, depending on their calculated mean value.

  • Aggregated Overall Risks Histograms: histograms that represent an overall aggregation of the frequency of all kinds of risks that affect suppliers, processes and storages of the model, depending on their calculated mean value.

  • Place of occurrence of Quantity/Quality/Delay Risk: pie charts that represent the share of each element concerned by this kind of risk

Monte-Carlo - Detailed charts report template

This report template consists in a detailed view of the risks impact on the elements of the model. It contains several global and detailed charts and detailed tables of the simulation results.

This kind of report contains 6 sections :

  • Risk Distribution: a pie chart that represents the share of each kind of risk (quantity, quality, delay) calculated through the queries written by the user.

  • System Risks Histograms: histograms that represent the frequency of each system risk (determined by its corresponding system query) depending on their calculated mean value.

  • Suppliers/Processes/Storages Risks Histograms: histograms that represent the frequency of each quantity/quality/delay risks that affect a given supplier/process/storage of the model, depending on their calculated mean value

  • Place of occurrence of Quantity/Quality/Delay Risk: pie charts that represent the share of each element concerned by this kind of risk

  • Main information about elements: table that contains detailed information about each element of the model.

  • Further information: table that contains results of the queries written by the user.

Furthermore, SimQRi allows the user to use its own report templates created with the Eclipse BIRT reporting tool ! For more information about this feature, please download and consult the PDF file that explains how to create your custom report template on the SimQRi tools webpage.
